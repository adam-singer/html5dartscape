Example of 3D landscape on HTML5 Canvas using Dart Editor. 
The goal was to see how quick and easy something in plain 
javascript could be converted to Dart. It is understood 
that current implemenation of dart compiles to a very 
large javascript file, again not what the goal of this was.
Original implementation can be found here: 
http://sebleedelisle.com/demos/html5landscape.html
